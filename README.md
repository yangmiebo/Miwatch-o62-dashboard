##纯前端表盘项目##
一个纯前端实现的智能手表表盘界面（Galaxy Watch 风格），模拟了时间显示、日期、电池电量环、天气、心率和步数等常用功能。
适合用于穿戴设备 UI 原型展示、前端练习或教学演示。

✨ 功能特性
实时时钟：显示时:分:秒，支持 AM/PM

日期显示：星期、月、日（英文大写）

电池电量环：SVG 环形进度条，颜色随电量变化（绿/黄/红）

电量百分比：顶部居中显示

天气温度：模拟数据（26°C）

心率模拟：每 3 秒随机生成 65~84 BPM

步数模拟：显示当前步数（可扩展真实数据）

深色太空背景：可自定义背景图片

发光圆环：配合电量进度的光晕效果

简洁底部指标：天气、心率、步数、秒/AM/PM 独立定位

🛠️ 技术栈
HTML5 + SVG（图形绘制）

CSS3：Flexbox、绝对定位、毛玻璃（可选）、阴影、圆角

原生 JavaScript：DOM 操作、Battery API、定时器、数据模拟

无需任何构建工具或依赖库，开箱即用。

🚀 如何运行
克隆或下载本项目到本地。

确保 assets/background.jpg 存在（或替换为自己的图片）。

用浏览器直接打开 index.html 文件（双击或在浏览器中打开）。

即可看到表盘界面，所有数据自动更新。

由于是纯前端，无需安装任何依赖，也无需启动服务器（但用 Live Server 等工具可以方便调试）。
